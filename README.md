# Smart Crime Tracker
A web-based crime tracking and management system built with **Python Flask**, designed to help users report, manage, and monitor crime-related information through a simple and responsive interface.

## Project Overview
**Smart Crime Tracker** is a full-stack web application that provides a platform for managing crime records efficiently. Citizens can file a report digitally and review its status. 
The project focuses on improving accessibility to crime reporting and tracking by providing a centralized digital solution.

---

## Features

* User authentication system
* Password hashing using **bcrypt**
* Crime tracking interface
* User account system
* Responsive web design
* Modern UI using **Bootstrap 5**
* Flask-powered backend

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/MdrBot/Smart-Crime-Tracker.git
```

### 2. Navigate into the project folder

```bash
cd Smart-Crime-Tracker
```

### 3. Create a virtual environment

```bash
python -m venv venv
```

### 4. Activate the virtual environment

**Windows:**

```bash
venv\Scripts\activate
```

### 5. Install dependencies

```bash
pip install bcrypt
pip install flask
```

### 6. Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000/
```
